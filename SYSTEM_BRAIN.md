## �️ TECH STACK REQUIREMENTS (LOCKED - NO EXCEPTIONS)

### Core Stack
- **Frontend:** Next.js 14+ (React) with TypeScript
- **Styling:** Tailwind CSS
- **Backend:** Next.js API routes / Server Actions
- **Forms:** React Hook Form + Zod validation
- **Database:** Optional for v1, structured PostgreSQL foundation for v2+
- **Hosting:** Vercel (optimized for Next.js)

### Non-Negotiable Requirements
✅ **Fully Responsive** - Mobile-first design, works on all devices  
✅ **Fast Load Time** - Lighthouse score 90+, LCP < 2.5s  
✅ **SEO-Friendly** - Metadata, Open Graph, schema markup  
✅ **Accessible** - WCAG 2.1 Level AA, basic ARIA labels  
✅ **Secure Form Handling** - Server-side validation, CSRF protection, input sanitization  

### What This Means
- Server-side form validation ALWAYS (never trust client-side alone)
- No sensitive data in client code
- HTTPS/SSL on all pages
- Rate limiting on API endpoints
- Email service via transactional provider (SendGrid/Resend)
- No database required for v1, but data structure must support PostgreSQL migration in v2
